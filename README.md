## Preview

```
https://estasleyendoesto.github.io/SolutionCSS/
```

## Installation

```bash
git clone https://github.com/Estasleyendoesto/solution-css.git
cd solution-css
npm install
```

## Uso

### Dev mode

```bash
npm run dev
```

### Build

```bash
npm run build
```
